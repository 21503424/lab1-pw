Lab 1

Neste report serão indicados os elementos observados na primeira aula, com o exemplo dos separadores apresentados ao fazer o inspect na nossa página:

  • Headers
  • Inspect
  • Preview
  • Timing

    HEADERS - indica-nos as ligações da nossa aplicação bem como a estrutura onde o código está inserido.

    INSPECT - mostra-nos o tamanho de cada ficheiro do projeto e o tempo de resposta ao carregamento de cada um.

    PREVIEW - dá-nos um overview daquilo que está criado até ao momento sem necessidade de executar todo o código para o resultado.

    PREVIEW - dá-nos de uma forma mais detalhada, o tempo que cada elemento necessita para ser executado bem com uma comparação entre o pedido e a resposta .
